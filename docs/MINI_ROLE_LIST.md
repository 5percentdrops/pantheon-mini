## Arthur's Special Responsibility

Arthur is the head of the Pantheon Mini.

Arthur replaces the default Paperclip CEO setup role with:

```text
Arthur = Project Manager / Head
```

Arthur is also responsible for hiring or activating extra agents if the mini team needs specialist help.

Extra agents are not active by default.
