# Mini Software House Policy

## Operating Principle

Use the smallest team that preserves safety:

```text
Arthur → Marcus → Jack → Cody → Marcus commit → Winston archive
```

Escalation only when needed:

```text
Jack fails → Marcus
Marcus fails → Maxwell
Maxwell fails or architecture breaks → Cody/Magnus
```

## Active Roles Only

Only these roles are active:

1. Arthur
2. Marcus
3. Jack
4. Cody
5. Maxwell
6. Magnus
7. Winston

## Jack Retry Cap

```text
Jack attempts = 1–12
```

After attempt 12 fails, Arthur executes Context Shredder and routes to Marcus.

## Marcus Attempts

```text
Marcus attempts = 13–15
```

Marcus gives tactical fixes.

## Maxwell Attempts

```text
Maxwell attempts = 16–17
```

Maxwell handles deep implementation rot.

## Cody Attempt

```text
Cody forensic review = attempt 18
```

Cody decides whether the failure is implementation-level or architecture-level.

## Magnus Attempt

```text
Magnus architecture rethink = attempt 19
```

Magnus proposes 1–3 structural pathways and terminates automation for manual review.

## Finalization

On Cody Pass:

```text
Arthur → Marcus final review → Marcus commit → Marcus PR draft → Arthur merge gate → Winston archive
```

## Merge

Merge remains gated.

```text
No silent auto-merge to main.
```

## Domain Specialists

Removed domain specialists must not be called in the mini version.

If a project truly needs frontend, mobile, Pine Script, Quantower, DevOps, or dedicated QA, Arthur must request manual approval to activate a specialist module.


## Arthur as Head

Arthur is the head and Project Manager of the Mini Software House.

In Paperclip setup terms, Arthur replaces the default CEO role.

```text
Arthur = Project Manager / Head
```

## Hiring Rule

Arthur is responsible for hiring or activating extra specialist agents if needed.

Specialists remain inactive unless Arthur produces a written hiring packet and the activation is approved or explicitly required by the project.

No hidden hires.
No silent specialist activation.
No routing to removed agents without a written reason.
