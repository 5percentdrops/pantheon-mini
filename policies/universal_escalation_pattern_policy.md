# Universal Escalation Pattern Policy

## Applies to
All active organisations.

## Harness
Hermes only.

Hermes must not be assigned as an agent harness.

## Model map
- Junior / Standard Developer: DeepSeek V4 Pro
- Senior Developer: Opus 4.7 X High
- Escalation Developer: Opus 4.7 Max
- Principal Engineer: Gemini Pro 3.1

## Attempt budgets
- Standard agent/developer: 15 self-fix attempts
- Senior owner: 3 solution attempts
- Maxwell / Staff Escalation Engineer: 2 solution attempts
- Cody / Code Review: 1 code/review pass
- Magnus / Principal Engineer: approach review

## Return routing
All higher-level returns route through the organisation router/project manager first.

## If worked
The standard agent continues.

## If failed
The router/project manager escalates according to remaining attempts and next layer.
