# DeepSeek V4 Pro Standard Developer Policy

## Rule
All standard / junior developer roles in this repo use DeepSeek V4 Pro under Hermes.

## Harness
Hermes only. OpenClaw must not be assigned as an agent harness.

## Applies to
- standard developers
- junior developers
- implementation developers
- execution developers
- QA engineers where they function as standard execution roles

## Does not apply to
- senior developers
- staff escalation engineers
- code reviewers
- principal engineers
- project managers
- research/strategy/advisory roles unless explicitly changed

## Model
```txt
DeepSeek V4 Pro under Hermes
```

Provider/model alias:
```txt
deepseek/deepseek-v4-pro
```
