# SoftwareHouse Harness Policy

Paperclip defines organisation, agents, routes, and assignments.
Hermes is the single harness — runs reasoning, planning, review, architecture, coding judgement, QA strategy, learning, and any deterministic/procedural tool work the agent needs. Blocked work escalates up the in-Hermes ladder; there is no second harness in V8.17+.

Model/module is not the job. The seed skill is the job manual.
