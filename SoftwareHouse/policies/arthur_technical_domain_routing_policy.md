# Arthur Technical Domain Routing Policy

Arthur must inspect every PRD/ticket and determine the implementation domain.

Arthur must not route only by market.

Required classification fields:
- target platform
- programming language
- runtime
- output artifact
- risk level
- specialist owner
- senior planner owner
- executor owner

Examples:
- TradingView/Pine Script → Felix + Ben
- Quantower/C# → Nathan + Grant
- Backend service/API → Marcus + Jack
