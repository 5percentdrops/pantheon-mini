# Arthur-Mediated Return Policy

## Rule
No higher-level engineer sends directly to Jack.

All returns from Marcus, Maxwell, Cody, or Magnus must pass through Arthur.

## Applies to
- Marcus solution attempts
- Maxwell solution attempts
- Cody code-review return packets
- Magnus approach-review reports

## Success handling
If Jack applies the returned solution and it works:

```txt
Jack marks WORKED
Jack continues task flow
```

## Failure handling
If Jack applies the returned solution and it fails:

```txt
Jack marks FAILED
Jack reports to Arthur
Arthur decides:
- same layer again if attempts remain
- next escalation layer if attempts are exhausted
```

## Magnus
Magnus remains approach-focused. This policy does not change Magnus's skill set.
