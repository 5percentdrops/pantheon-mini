# Error Memory Logging Ownership Policy

## Rule
Whoever proposes the solution logs the proposed solution.

Jack / the standard developer logs the blocker and the tested result.

Arthur verifies that the log is complete before routing the issue forward.

## Single-folder rule

```txt
wiki/errors/
```

This is the only folder for blocker, solution, code-fix, and approach-solution logs.

## Allowed log types
- BLOCKER_LOG
- SOLUTION_LOG
- CODE_FIX_LOG
- APPROACH_SOLUTION_LOG

## Not allowed
- general notes
- PRDs
- SDDs
- feature tickets
- random scratchpads
- unrelated research
