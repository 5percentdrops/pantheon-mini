# Hermes Harness Runtime Policy

Hermes is the only standard agent harness for Pantheon Mini V8.

Hermes sits on top of the selected LLM model. The model may be GPT-5 mini, GPT-5.5, Opus 4.7 XHigh, Opus 4.7 Max, Gemini Pro 3.1, DeepSeek V4 Pro, or another configured provider model.

Hermes supplies:

- persistent memory
- skills
- session continuity
- tools
- MCP integration
- model-provider execution
- recurring routines

Hermes does not replace Paperclip. Paperclip is the company/control plane.
Hermes does not replace the model. The model is the reasoning engine inside Hermes.
