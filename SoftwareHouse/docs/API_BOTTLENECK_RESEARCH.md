# API & Bottleneck Research

## Owner
Vera — API & Bottleneck Intelligence Agent

## Purpose
Before feasibility is judged, Vera maps APIs, data sources, free tiers, paid tiers, rate limits, and bottlenecks.

## Required API/source checks
- what the API/source is
- free package availability
- free package limitations
- paid package cost
- paid package limits
- rate limits
- requests per second/minute/day/month
- auth requirements
- data available
- data missing
- scale risks
- security risks
- TOS/policy risks
- alternatives
- custom API/wrapper option

## Decision principle
Free is not automatically best.
Paid is not automatically best.
Best equals lowest total bottleneck cost.
