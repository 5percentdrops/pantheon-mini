# Superpowers TDD for Senior Engineers

## Rule
Every senior engineering owner must use the Superpowers TDD skill when converting feature tickets into executable task blocks.

## TDD format

For each task:

```txt
Red test / failing check
→ minimal implementation
→ green passing check
→ acceptance criteria
→ PR criteria
```

## Purpose
This prevents hallucinated implementation, skipped steps, vague tasks, and untested feature work.
